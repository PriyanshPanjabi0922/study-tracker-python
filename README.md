# 📚 Study Tracker (Python)

A command-line Study Tracker built in Python that helps users record, manage, and review their study sessions. The project stores data using JSON, allowing study sessions to persist even after the program is closed.

## Features

- Add a study session
- View all study sessions
- Search sessions by subject
- Calculate total study hours
- View today's study sessions
- Delete a study session
- Save data using JSON
- Load previous study sessions automatically

## Technologies Used

- Python 
- JSON
- datetime module

1. Clone the repository.
2. Open the project folder.
3. Run:
